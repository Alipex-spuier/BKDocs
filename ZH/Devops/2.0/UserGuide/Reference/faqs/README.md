## 故障排查流程自助：

1. FAQ无法找到解决方案，请按如下要求提供反馈信息：

   1.1  提供 BK-CI 版本号（BK-CI 首页可获取）

   1.2  问题现象描述及报错截图（是否必现，操作内容，配置截图等）

   1.3  提供构建日志   (获取方式参照 [基本概念](user-guide.md))

   1.4  提供服务日志   （可选）
