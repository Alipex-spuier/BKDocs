```bash

```

## 5. 开始更新

### 5.1. 管控平台

```bash
./bkcli sync gse
./bkcli upgrade gse

# 检查 GSE 服务
./bkcli status gse
./bkcli check gse
```

### 5.2. 配置平台

```bash
./bkcli upgrade cmdb
./bkcli status cmdb 
./bkcli check cmdb
```

### 5.3. 标准运维

```bash
./bk_install saas-o bk_sops
```

### 5.4. 流程服务

```bash
./bk_install saas-o bk_itsm
```
