# 产品简介


## 产品概述

蓝鲸消息通知中心（BKNotice）是一套蓝鲸PaaS平台的基础服务，旨在提供给平台管理者和应用开发者一个站内的消息通知渠道，如：平台公告、活动通知、个人消息等等
